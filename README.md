# OpenClaw Skill: jenkins

> This is an agent skill for [OpenClaw](https://openclaw.ai).

Interact with Jenkins CI/CD server via REST API.

## Installation

```bash
clawhub install guoway/openclaw-skill-jenkins
```

## Configuration

Set environment variables:
- `JENKINS_URL` - Your Jenkins server URL
- `JENKINS_USER` - Your username
- `JENKINS_API_TOKEN` - Your API token
